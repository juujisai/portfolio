Nazywam się Bartosz, jestem absolwentem UWM w Olsztynie. Posiadam tytuł magistra inżyniera gospodarki przestrzennej lecz chciałbym spróbować swoich sił w innej dziedzinie. Już na studiach zajęcia związane z programowaniem wydawały mi się najbardziej interesujące i właśnie dlatego postanowiłem rozpocząć naukę web-developementu. Uczę się z przerwami prawie rok. Projekty, które przedstawione zostaną poniżej powstały w dużej mierze z ciekawości i podstawowego pytania: czy jestem w stanie to zrobić? Zapraszam!

Zaplecze techniczne

Javascript

JavaScript

Przy tworzeniu stron internetowych i programów wykorzystuję język JavaScript w standardzie ECMAScript 6.

HTML

HTML

Język znaczników HTML wykorzystuję w wersji HTML 5.

CSS

CSS

Do nadania styli moim stronom wykorzystuję nowości, które wprowadzone zostały w trzeciej wersji specyfikacji CSS (CSS3).

React

React

Już od jakiegoś czasu staram się praktykować moją przygodę z biblioteką React.js, wraz ze środowiskiem create-react-app oraz biblioteką react-router-dom. Jest to świetne narzędzie do tworzenia Single Page Application (SPA)

OpenLayers

OpenLayers

OpenLayers umożliwia przedstawienie danych Systemu Informacji Przestrzennych GIS. Biblioteka oparta jest na języku JavaScript i pozwala na tworzenie interaktywnych portali mapowych.

GIMP

GIMP

Jest to oprogramowanie, które wykorzystuję do edycji zdjęć i obrazów rastrowych. Jest to wygodne oprogramowanie dostępne bezpłatnie.

Figma

Figma

Figma służy mi za miejsce do opracowywania danych wektorowych. W oprogramowaniu tym tworzę swoje wszystkie ikony, które później wykorzystuję na stronach internetowych.
